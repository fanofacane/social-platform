## 技术栈
- SpringBoot
- SpringAI
- Netty
- Mysql
- Mybatis
- Redis
- MinIo

## 项目启动
1. npm install
2. npm run dev

## 功能
- 聊天实时通信+消息提醒
- MinIO对象存储
- feed流关注推送+滚动分页
- AI客服检索查询视频
- 短视频的点赞收藏关注等基础功能
